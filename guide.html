快速开始

五 分 钟 内 启 动 你 的 第 一 个 案 件
QUICK START
GET UP AND RUNNING
STEP 01

克隆项目并安装依赖

git clone git@github.com:xiaomutou-git/KnowledgeDetective.git cd KnowledgeDetective pnpm install
STEP 02

配置 AI API Key

cp .env.example .env # 编辑 .env 文件，填入你的 API Key AI_API_KEY=your-api-key-here

推荐使用 硅基流动 SiliconFlow，新用户赠送 2000 万 Token。

STEP 03

启动本地代理服务器

node proxy-server.js

服务默认运行在 http://localhost:4315。

STEP 04

打开浏览器开始推理

访问首页 http://localhost:4315/，点击「开始推理」进入游戏页面，选择案卷或上传自己的材料。

操作指南

从 开 卷 到 封 存 的 完 整 流 程
DETAILED OPERATIONS
CASE WORKFLOW
01

选择案卷

进入推理页面后，可以从精选案卷中选择一个主题，或滚动到页面底部使用「上传你的卷宗」功能。

02

阅读场景

仔细阅读案件背景故事。场景会逐步揭示关键信息，帮助你将抽象概念嵌入到具体情境中。

03

收集线索

点击线索卡片查看详细信息。所有线索收集完毕后，才能对推理问题做出判断。

04

回答问题

根据场景与线索，选择你认为正确的选项。提交后会立即显示答案与详细解析。

05

查看解析

解析包含正确答案、关键要点、公式推导与知识拓展。即使答错，也能从解析中完成学习。

06

封存档案

点击「封存卷宗」生成知识卡。知识卡会自动保存到浏览器的本地档案库，方便随时复习。

环境配置

服 务 启 动 前 的 准 备 工 作
ENVIRONMENT SETUP
REQUIREMENTS & CONFIG
req

系统要求

Node.js >= 14.0.0 · 现代浏览器（Chrome / Edge / Firefox / Safari）· 网络连接（用于调用 AI API）

env

环境变量说明

在 .env 文件中配置以下变量：

AI_API_KEY=your-api-key-here AI_ENDPOINT=https://api.siliconflow.cn/v1/chat/completions AI_MODEL=Qwen/Qwen2.5-7B-Instruct
run

启动命令

node proxy-server.js

启动成功后，控制台会打印访问地址与 API 状态。如果 API Key 未配置，服务器会提示并退出。

常见问题

FAQ · 解 答 你 的 疑 惑
FREQUENTLY ASKED
QUESTIONS

本地服务启动后，前端页面可以直接在浏览器中打开。但如果你使用「自定义卷宗」功能让 AI 生成剧本，则需要联网调用 AI API。

不会。API Key 仅存储在服务器的环境变量中，由 proxy-server.js 代理调用 AI API。前端只与本地代理通信，不会直接接触 API Key。

当前支持直接粘贴文本、上传 TXT 文件、PDF 文件（依赖 pdf.js）以及 Word 文档（依赖 mammoth.js）。建议文本长度不少于 50 字，以获得更好的生成效果。

所有案件数据与知识卡都保存在浏览器的 localStorage 中，不会上传到服务器。清理浏览器数据会导致本地档案丢失，请知悉。

请检查：1) API Key 是否正确配置；2) 网络连接是否正常；3) 输入文本是否过短或包含特殊格式。你也可以通过 /api/health 接口检查服务器与 AI 配置状态。
