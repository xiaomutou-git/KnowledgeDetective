# 贡献指南

欢迎贡献代码！请遵循以下指南。

## 提交代码

### 提交规范

请使用 [Conventional Commits](https://www.conventionalcommits.org/) 格式：

- `feat`: 新功能
- `fix`: 修复问题
- `docs`: 文档更新
- `style`: 代码风格（不影响功能）
- `refactor`: 代码重构
- `test`: 测试
- `chore`: 构建或工具更新

### Pull Request 流程

1. Fork 项目
2. 创建分支：`git checkout -b feature/xxx`
3. 提交代码：`git commit -m "feat: 新增功能"`
4. 推送到远程：`git push origin feature/xxx`
5. 创建 Pull Request

## 代码规范

- 使用 JavaScript ES6+ 语法
- 代码注释完整
- 保持代码风格一致
- 避免使用不安全 API

## 开发环境

1. 安装 Node.js >= 14.0.0
2. 安装依赖：`npm install`
3. 启动开发服务器：`npm start`
4. 运行测试：`npm test`

## 报告问题

请在 [Issues](https://github.com/xiaomutou-git/KnowledgeDetective/issues) 中报告问题，并提供详细信息。

## 许可证

本项目采用 MIT 许可证，贡献代码即表示同意该许可证。
